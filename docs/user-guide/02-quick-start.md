# 快速开始

本指南将帮助您在几分钟内开始使用 Smart Draw。



## 第一步：配置 AI 服务

首次使用需要配置 AI 服务。有两种方式：

### 方式一：使用自己的 API Key（推荐个人用户）

1. 点击聊天面板右上角的 **设置图标** ⚙️
2. 选择 **本地配置** 模式
3. 填写配置信息：
   - **服务商**：选择 OpenAI 或 Anthropic
   - **API Key**：填入您的 API 密钥
   - **模型**：选择要使用的模型（推荐 Claude Sonnet 4.5）
4. 点击 **保存**

### 方式二：使用访问密码（推荐团队用户）

1. 点击聊天面板右上角的 **设置图标** ⚙️
2. 切换到 **访问密码** 模式
3. 输入管理员提供的访问密码
4. 点击 **验证** 完成配置



## 第二步：选择绘图引擎

在聊天面板顶部的下拉菜单中选择：

| 引擎 | 风格 | 适用场景 |
|------|------|----------|
| **Draw.io** | 专业、规整 | 正式文档、技术方案、企业汇报 |
| **Excalidraw** | 手绘、随性 | 头脑风暴、创意设计、非正式演示 |


## 第三步：描述您的图表

在输入框中用自然语言描述您想要的图表，例如：

```
画一个电商下单流程图，包含：
1. 用户浏览商品
2. 加入购物车
3. 提交订单
4. 支付
5. 订单完成
```



## 第四步：生成并使用

1. 点击 **发送** 按钮
2. 等待 AI 生成（可实时查看生成过程）
3. 生成完成后点击 **应用到画布**
4. 在画布上查看、编辑您的图表
